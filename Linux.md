### Change Owner of a file

```shell
sudo chown -R username:group directory
```

example

```shell
sudo chown -R ubuntu:ubuntu
```



### Check Current Directory

```shell
pwd
```



### Change File Permissions

```shell
chmod 
```

Example

```shell
chmod 400 sample_server.pem
```
